# goodreads_books_sql_analysis


This project analyzes a Goodreads dataset I downloaded — around 13K rows with authors, titles, years, pages, genres, ratings, and review counts. I created the table in PgAdmin, then imported the data from the CSV into pgAdmin as the goodreads_works table, then wrote queries to dig in to the data and see what could be determined.


The 10 queries cover some basic to mid-level skills: counting distinct authors, top producers, lowest-rated popular books, hidden gems, publication era splits, longest books, and authors active 30+ years. Each .sql file starts with numbered name and comments on what it does and key findings.


Load the data and run the files to follow along.
